# 关于mox
mox是基于PHP+MYSQL的社区+电商系统，支持PC和移动端。目前主要功能模块有社区和电商两大功能，主要是想通过聚焦用户的兴趣，以兴趣为场景，搭建出一个社区环境，在这个环境里嵌入了兴趣相关的周边商店。
社区模块包含关注、发布动态、朋友圈、话题、个人主页、积分、点赞、评论等社区基本功能。
电商模块包含商品、购物车、订单、分销、拼购等电商基本功能。
