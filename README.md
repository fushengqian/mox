# 关于mox
mox是基于PHP+MYSQL的免费使用社区，支持PC和移动端。主要功能模块有关注、发布动态、朋友圈、话题、个人主页、积分、点赞、评论等社区应有的功能。作者QQ：540335306。本项目将持续进行完善，敬请期待！

# 前台主要系统功能模块
1）关注</br>
2）动态</br>
3）话题</br>
4）评论</br>
5）点赞</br>
6）朋友圈</br>
8）个人主页</br>
9）消息盒子</br>
10）用户中心</br>
11）登录注册</br>

# 后台管理系统功能模块
1）用户管理</br>
2）动态管理</br>
3）话题管理</br>
4）评论管理</br>
5）系统管理</br>
